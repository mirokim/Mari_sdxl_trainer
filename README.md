# Mari_sdxl_trainer

실행 방법
백엔드:

cd C:\dev\Mari_sdxl_trainer\backend

pip install -r requirements.txt

python main.py

프론트엔드:

cd C:\dev\Mari_sdxl_trainer\frontend

npm install

npm run dev


http://localhost:3000에서 노션 스타일 UI에 접속, http://localhost:8000에서 API가 실행됩니다. WebSocket으로 학습 진행 상태가 실시간 전송됩니다.
